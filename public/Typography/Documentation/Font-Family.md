# TYPOGRAPHY TAILWINDCSS

## Font Family TailwindCSS

```css
font-sans
font-serif
font mono
```

## Configurasi Custom

Setting file tailwind.config.js

```js
module.exports = {
  theme: {
    fontFamily: {
      fontFamily: {
        custom: ["Avenir", "times new roman", "sans-serif"],
      },
    },
  },
};
```
